# Cognifyz Technologies Software Development Internship

This repository contains the projects completed during my Software Development Internship at Cognifyz Technologies.

## Internship Tasks Completed

### Task 1 - Number Guessing Game
A console-based number guessing game developed using Python.

**Features**
- Random number generation
- User input handling
- Hints for higher/lower guesses
- Unlimited attempts

---

### Task 2 - Number Pattern Generator

A menu-driven application to generate different number patterns using loops.

**Features**
- Right Triangle
- Inverted Triangle
- Number Pyramid
- Diamond Pattern

---

### Task 3 - Console Task Manager

A CRUD-based task management system.

**Features**
- Add Task
- View Tasks
- Update Task
- Delete Task
- Mark Task Completed
- Statistics

---

### Task 5 - Persistent Task Manager

An advanced version of Task Manager with file storage.

**Features**
- CRUD Operations
- File Handling
- Data Persistence
- Automatic Task Loading
- Error Handling

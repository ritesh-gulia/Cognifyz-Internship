# Number Guessing Game

## 📌 Description
This is a simple text-based Number Guessing Game developed using Python as part of the Cognifyz Technologies Software Development Internship.

## 🚀 Features
- Three difficulty levels
- Random number generation
- User-friendly interface
- Invalid input handling
- Displays total attempts

## 🛠️ Technologies Used
- Python 3

## ▶️ How to Run

```bash
python main.py
```

## 📷 Sample Output

```text
Choose Difficulty:
1. Easy
2. Medium
3. Hard
```

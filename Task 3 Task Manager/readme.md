# Task 3 - Console Based Task Manager

## Objective
Develop a console-based Task Manager application using Python that performs CRUD (Create, Read, Update, Delete) operations.

## Features
- Add New Task
- View All Tasks
- Update Existing Task
- Delete Task
- Mark Task as Completed
- Show Task Statistics
- Menu Driven Interface
- Input Validation

## Technologies Used
- Python 3

## How to Run

```bash
python3 main.py
```

## Learning Outcomes
- Lists
- Dictionaries
- Functions
- Loops
- Conditional Statements
- Exception Handling

# Task 5 - Persistent Task Manager

## Objective
Develop a console-based Task Manager with CRUD operations and persistent file storage using Python.

## Features
- Add Task
- View Tasks
- Update Task
- Delete Task
- Mark Task as Completed
- Show Statistics
- Persistent Storage using `tasks.txt`
- Automatic Loading of Saved Tasks
- Input Validation
- Menu-Driven Interface

## Technologies Used
- Python 3

## Files
- `main.py` - Main application
- `tasks.txt` - Stores task data
- `README.md` - Project documentation

## How to Run

```bash
python3 main.py
